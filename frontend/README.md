# 🛡️ Aegis SOC Dashboard (Frontend)

This is the frontend component for **Aegis – Intelligent SIEM & Threat Response Platform**.
It provides a production-grade, real-time Security Operations Center (SOC) dashboard built with **Next.js 14**, **React 19**, and **Tailwind CSS**.

The interface is designed with a **Cyberpunk Experience** in mind—featuring a fully immersive, sci-fi-themed UI with holographic effects, retro-terminal logs, and real-time system status indicators.

## 🚀 Key Features

- **Real-Time Monitoring**: Live feed of Logs, Alerts, and Incidents via a modern UI.
- **Cyberpunk Aesthetic**: Dark mode by default, neon accents, and dynamic animations.
- **Investigation & Forensics**: Deep dive into raw logs with enrichment context (GeoIP, User-Agent), visualized using interactive maps.
- **Advanced Data Visualization**: Real-time charts and metrics powered by Recharts.
- **Fluid Animations**: Smooth transitions and micro-interactions utilizing Framer Motion.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **UI & Styling**: [React 19](https://react.dev/), [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Maps**: [Leaflet](https://leafletjs.com/) & [React Leaflet](https://react-leaflet.js.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## ⚡ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Navigate to the `frontend` directory (if not already there):

```bash
cd frontend
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

Standard Next.js App Router structure:

- `app/`: Contains the main application routes, layout, and global styles.
- `public/`: Static assets deployed along with the application.
